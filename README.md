# TextUtils - Text Manipulation Utility

A fast and intuitive web application for performing various text operations and transformations. TextUtils makes it easy to manipulate, analyze, and transform text content with a clean, user-friendly interface.

## 🌟 Features

- **Text Transformation**: Convert text to uppercase, lowercase, and capitalize options
- **Text Analysis**: Get character count, word count, and reading time estimates
- **Copy to Clipboard**: Easily copy processed text with one click
- **Clear Text**: Quick reset functionality to start fresh
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Dark Mode Support**: Optional theme switching for comfortable viewing
- **Real-time Preview**: See changes instantly as you work

## 📋 Components

- **Navbar**: Navigation bar with branding and theme toggle
- **TextForm**: Main text input and transformation interface
- **Alert**: User feedback and notifications system
- **About**: Information page about the application

## 🚀 Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/textutils.git
```

2. Navigate to the project directory:

```bash
cd textutils
```

3. Install dependencies:

```bash
npm install
```

### Available Scripts

In the project directory, you can run:

#### `npm start`

Runs the app in development mode.

- Open [http://localhost:3000](http://localhost:3000) to view it in your browser
- The page will reload when you make changes
- Console will display lint errors

#### `npm run build`

Builds the app for production to the `build` folder.

- Correctly bundles React in production mode
- Optimizes the build for best performance
- Build is minified with hashed filenames
- Ready for deployment

#### `npm test`

Launches the test runner in interactive watch mode.

## 💡 Usage

1. Start the development server with `npm start`
2. Enter or paste text in the text form
3. Use the available buttons to transform your text:
   - Convert to uppercase/lowercase
   - Capitalize text
   - Clear all text
4. Copy the result to your clipboard with the copy button
5. Use the navigation bar to access additional features
